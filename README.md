## About

This portfolio highlights my skills and experience in Quality Assurance (QA) and software testing. It showcases my approach to ensuring software quality through test artifacts, techniques, and process expertise. The project is designed to present my work, methodologies, professional focus, and contact details for prospective employers and collaborators in the QA domain.

## Features

- Accueil: Welcome banner
- À propos: A section detailing professional background and interests in QA
- Compétences: List of QA/testing skills and expertise
- Portfolio: Showcase of QA/tester projects and achievements
- Contactez-moi: Means to get in touch for collaboration or professional inquiries

## Technology Stack

This portfolio site uses:

- **HTML5** (`index.html`)
- **CSS3** (`style.css`)
- **JavaScript** (files in `/script`)
- Directory for images: `/images`
- Data folder: `/data`

No framework dependencies—this is a pure web portfolio.

## Getting Started

To view or edit the portfolio locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/miggiu/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your preferred web browser.
